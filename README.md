## Installing & Testing

### TL;DR

```sh
git checkout git@github.com:PracticaDS/pdes-tp-e2e-anonymous.git
cd pdes-tp-e2e-anonymous
yarn install
echo '{ "host": "https://practicads.github.io/pdes-tp-anonymous/" }' > cypress.env.json
yarn cypress:open
```
